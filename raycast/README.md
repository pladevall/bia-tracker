# Raycast - Baseline

Raycast extension to add actions and beliefs quickly.

## Setup
1. Open Raycast and add a local extension from `raycast`.
2. Set preferences:
   - API Base URL: `https://bia-tracker-pladevalls-projects.vercel.app`
   - API Token: optional (only if your API requires auth)

## Commands
- Add Action: creates a bold take (action)
- Add Belief: creates a belief

## Notes
- Actions can link to a belief or a bet.
- Beliefs can link to a bet.
